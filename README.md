# Install di VPS
1.
```
sudo apt update && sudo apt upgrade -y
```
2.
```
sudo apt install git -y
```
3.
```
sudo apt install python3 python3-pip -y
```
4. 
```
git clone https://github.com/iyayn-ajah/tools-spam-web-qna
```
5.
```
pip3 install requests
```
6.
```
cd tools-spam-web-qna
```
7.
```
python3 aqaspam.py 
```
atau
```
python3 nglspam.py
```
